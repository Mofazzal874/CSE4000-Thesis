Fig: Yolo11 Architecture
                            Changes in the Architecture

       Model         Backbone Change      Head change               Hypothesis

YOLO11m Baseline     Standard (C2PSA)     3-scale (P3, P4, P5)      Reference model

YOLO11m + CBAM       CBAM replaces C2PSA  3-scale (P3, P4, P5)      Better feature selection via
                                                                                attention

YOLO11m + P2Head     Standard (C2PSA)     4-scale (P2, P3, P4, P5)  Higher resolution for tiny
                                                                                 objects

Yolo11m+CBAM+P2Head  CBAM replaces C2PSA  4-scale (P2, P3, P4, P5)  Best of both: attention +
                                                                             resolution

Notes

· CBAM: Replaces built-in C2PSA Attention Layer with CBAM(channel +spatial

attention , but seqential)

· P2 Head:Added 3 extra layers to the detection head to create a 4th detection scale

at 160×160 resolution (stride 4). This means each detection grid cell covers only 4×4             11

pixels, allowing the model to "see" objects as small as 4px.
